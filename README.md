# 🛒 E-commerce AI Agent (NLP → SQL)

This project is an AI-powered system that allows users to query a database using natural language.

## 🚀 Features

* Convert text → SQL using Amazon Bedrock (Claude)
* Execute SQL on SQLite database
* Return clean, human-readable answers
* Handles fuzzy search (LIKE, case-insensitive)
* Streamlit UI

## 🧠 Tech Stack

* Python
* AWS Bedrock
* SQLite
* Streamlit

## ▶️ How to Run

```bash
git clone <your-repo-link>
cd Ecommerce-Ai
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python create_ecommerce_db.py
streamlit run app.py
```

## 🧪 Example Queries

* Which product is most expensive?
* Price of headphone
* What did Rishu order?
